# ChatPayShield

AI-powered fraud detection and transaction risk intelligence system designed to analyze suspicious payment activity, behavioral anomalies, scam language, and transaction trust signals.

## Core AI Features

* Scam NLP analysis
* Behavioral anomaly detection
* Transaction risk scoring
* OCR-based screenshot analysis
* Explainable AI trust scoring

## Tech Stack

### Backend

* Node.js
* Express.js
* MongoDB

### AI

* NLP
* OCR
* Risk Analysis Engine

### Frontend

* React
* Tailwind CSS
